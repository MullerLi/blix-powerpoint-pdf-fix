# Blix 衍生修改聲明 / Modification Notice

Blix（卜力士）是基於 IBM Plex Sans TC 與 cathree3/Plix 上游專案的改名衍生版本。

Blix (卜力士) is a renamed derivative based on the IBM Plex Sans TC and
cathree3/Plix upstream projects.

- 上游衍生專案 / Upstream derivative project: <https://github.com/cathree3/Plix>

## 本版本修改 / Modifications in This Build

- 修復 Windows BMP `cmap` format 4 對照表。
- 改善 PowerPoint 字型內嵌與 PDF 輸出相容性。
- 正規化 PDF 字重 metadata。
- 更新版本與 Unique ID，以降低 Windows 舊字型快取衝突。

- Repaired Windows BMP `cmap` format-4 mappings.
- Improved PowerPoint embedding and PDF export compatibility.
- Normalized PDF weight metadata.
- Updated revisions and unique IDs to reduce stale Windows font-cache
  conflicts.

## 名稱與協議 / Naming and License

本衍生版本不使用原始 Reserved Font Name `Plex`，而使用 `Blix / 卜力士`。
修改版本仍完整適用 SIL Open Font License 1.1，且未新增其他 Reserved Font Name。

This derivative does not use the original Reserved Font Name `Plex`; it uses
`Blix / 卜力士`. The modified font remains entirely under the SIL Open Font
