# IBM Plex Sans TC 來源修改聲明 / Derivative Notice

本專案公開散布的 **INK Sans TC（墨黑體）** 是 IBM Plex Sans TC
的 PowerPoint / PDF 相容修正版。

The publicly distributed **INK Sans TC** family, localized as **墨黑體**, is a PowerPoint / PDF
compatibility-modified derivative of IBM Plex Sans TC.

## 本版本修改 / Modifications in This Build

- 修復 Windows BMP `cmap` format 4 對照表。
- 改善 PowerPoint 字型內嵌與 PDF 輸出相容性。
- 正規化 PDF 字重 metadata。
- 更新版本、Unique ID 與 checksum。
- 將內部 family 與 PostScript name 改為 `INK Sans TC` / `INKSansTC`，以遵守原始協議的
  Reserved Font Name 條款。

- Repaired Windows BMP `cmap` format-4 mappings.
- Improved PowerPoint embedding and PDF export compatibility.
- Normalized PDF weight metadata.
- Updated revisions, unique IDs, and checksums.
- Renamed the internal family and PostScript names to `INK Sans TC` /
  `INKSansTC` to
  comply with the original agreement's Reserved Font Name clause.
