# 授權文件 / License Documents

本目錄分別列出 Blix 與 IBM Plex Sans TC 適用的 SIL Open Font License 1.1。

The applicable license for both Blix and IBM Plex Sans TC is the SIL Open Font
License 1.1.

## 文件 / Documents

- `Blix-SIL-OFL-1.1.txt`：Blix 適用的 SIL OFL 1.1 全文。
- `Blix-MODIFICATION-NOTICE.md`：Blix 衍生來源與修改聲明。
- `IBM-Plex-Sans-TC-SIL-OFL-1.1.txt`：IBM Plex Sans TC 官方 SIL OFL 1.1 全文。

IBM 原始協議將 `Plex` 指定為 Reserved Font Name。Blix 是重新命名後的衍生版本，
未新增其他 Reserved Font Name。

IBM's original agreement reserves the name `Plex`. Blix is a renamed
derivative and declares no additional Reserved Font Names.
